\# GreenGrocerSystem



A PHP \& MySQL based Green Grocer Market Management System.



\## Features

\- Admin dashboard

\- Product \& order management

\- Customer \& warehouse management



\## How to Run

1\. Install XAMPP

2\. Start Apache \& MySQL

3\. Place project in htdocs

4\. Import greengrocer.sql

5\. Open http://localhost/GreenGrocerSystem
